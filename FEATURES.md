# 功能需求跟踪

> 按时间倒序记录新增功能点，完成后标记 ✅

## v0.2.0 — 文档管理增强（已完成 ✅）

### 批量上传
- [x] 拖拽多文件后显示上传队列
- [x] "全部上传"按钮触发批量上传
- [x] 单文件独立进度，失败不影响其他文件
- [x] 上传完成后可取消队列

### 文件去重
- [x] 后端基于 `filename + file_size` 检测重复
- [x] 重复文件自动加后缀（`file_1.docx`、`file_2.docx`...）作为新记录入库
- [ ] （可选）基于 SHA-256 内容哈希精确去重

### 删除功能
- [x] 后端 `DELETE /files/{id}` — 同时删除 MinIO 对象和 DB 记录
- [x] 文档列表页每行显示删除图标
- [ ] 文档详情页增加删除按钮
- [x] 删除前弹窗确认

### 批量操作
- [x] 文档列表多选复选框
- [x] 批量删除选中文件
- [x] 批量重新解析选中文件（针对失败/需要重新解析的文档）
- [x] 操作栏显示"已选择 X 个文件"

### 实时状态更新
- [x] pending/processing 状态的文档自动轮询（每 3s）
- [x] 解析完成后自动刷新列表，无需手动刷新
